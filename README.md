# GitHub Pages ノーコードサイトビルダー

## 構成
- `index.html`
- `assets/css/app.css`
- `assets/js/app.js`
- `assets/js/canvas.js`
- `assets/js/code-editor.js`
- `assets/js/components.js`
- `assets/js/export.js`
- `assets/js/history.js`
- `assets/js/inspector.js`
- `assets/js/logic.js`
- `assets/js/storage.js`
- `assets/js/templates.js`
- `assets/js/utils.js`

## 使い方
1. `index.html` を GitHub Pages で公開
2. 左ペインのパーツを追加
3. 中央で移動・リサイズ
4. 右ペインでプロパティ、コード、ロジック、設定を編集
5. ZIP / HTML / JSON を出力
6. GitHub デプロイ画面から直接 push を試行

## 補足
- モバイルでは自動で縦レイアウトに切り替わります
- 保存内容は LocalStorage に自動保存されます
- HTML / CSS / JS の分割出力と 1ファイル出力の両方に対応しています
